# XYLO Web (FE)

## 개발환경

# 환경

- 에디터
  - `VSCode`
- VSCode 플러그인
  - `Prettier - Code Formatter`
  - `Eslint`
- 패키지 매니저
  - [yarn](https://classic.yarnpkg.com/)
- 커밋 메세지 subject (예시: `feat: 로그인 페이지 추가`)
  - chore: 빌드 업무, 패키지 매니저 수정
  - docs: 문서 수정
  - feat: 새로운 기능 추가
  - fix: 버그 수정
  - perf: 성능 개선
  - refactor: 리펙토링
  - revert: 기능 되돌리기
  - style: 스타일 수정
  - test: 테스트 수정

## 설치

```shell
yarn
```

## 실행

```shell
yarn start
```

## 빌드

```shell
yarn build
```
